Визуальный язык построен вокруг метафоры скатерти-самобранки: тёплый кремовый холст, золотые акценты и глубокие восточные цвета создают сцену, на которой главный герой — блюдо.

Система оптимизирована под мобильный web. Интерфейс остаётся спокойным и тихим, чтобы фотографии блюд вид сверху звучали в полный голос.

принципы

  1. фото — герой. интерфейс — рама. никогда не конкурируем с тарелкой.
  2. кремовый холст. белых пустот нет — есть тёплая скатерть и её оттенки.
  3. золото — нить. тонкие орнаменты, точки, окантовки; никогда не заливки.
  4. цвет — щедрость. красный, зелёный, индиго — насыщенно, но редко.
  5. типографика. alice — для блюд и цен. roboto flex — для всего остального.

структура
  1. grab handle. 44×4 px, ink-200.
  2. hero без подложки. блюдо на 120% ширины выезжает за правый край, медленно вращается (120 с полный оборот) — взгляд утыкается в еду.
  3. павлин-фон. opacity 16%, позади блюда — ненавязчивый орнамент.
  4. title + price. alice 24 / alice 24 wine. один ряд.
  5. описание. Roboto Flex 13, ink-700, с прописной.
  6. ползунок веса. без подложки, прямо на фоне шита.
  7. опции. pills, toggle.
  8. cta. secondary + green на всю ширину.
